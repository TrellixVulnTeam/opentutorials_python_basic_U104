 < 인캡슐레이션 : 캡슐화 >

 - 객체 == 부품
 - 변수들은 함수라는 케이스(객체)로 한대 묶여 사용된다. 이를 캡슐화 라고한다.
 - 객체 지향프로그램의 중요한 원칙중 하나이다.
 - 목적은, 객체 안에 포함되어있는 데이터가 외부로 부터 영향을 받지 않게 하기위해 만들어진 것이다.
 
 데이터 vs 로직
 => HDD vs CPU
 => 하드에 데이터가 저장되어있다면 하드에 저장된 데이터의 가치가 더욱 클 것이다.
 => 데이터 즉, 변수란 커다란 가치를 가지고있다.
 => 따라서 이러한 데이터의 외부영향을 막기위해 캡슐화는 중요하다.
